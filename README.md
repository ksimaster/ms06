# DigitShooterClone

Bu projede Digit Shooter oyununun 3 leveldan oluşacak şekilde clone versiyonunu yaptım. Level arası data kaybını önlemek için Scriptableobject kullandım. Bu sayede para ve booster levelları seviyeden seviyeye aktarılmış oldu.
